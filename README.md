# Fitness Calendar App

## Project Goals

The Fitness Calendar App is designed to help users track their fitness, nutrition, and wellness habits in one organized place. Users can log daily workout and nutrition details, making it easier to monitor consistency, progress, and overall health goals over time.

The main goal of this project is to create a simple calendar-based fitness tracker where users can record important daily health data, including:

- Workout type
- Workout length
- Calories consumed
- Protein intake
- Carbohydrate intake
- Water intake
- Fat intake

This app helps users stay accountable by giving them a clear view of their fitness routine and nutrition habits.

## Features

- Add workout entries to specific calendar dates
- Track workout type and workout duration
- Record daily calorie intake
- Track macronutrients: protein, carbs, and fat
- Track daily water intake
- View fitness and nutrition data by date
